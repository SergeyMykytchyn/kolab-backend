Все запросы к серверу начинаются так: https://kolab-plumboomteam-api.herokuapp.com/api
Дальше идет название определенного эндпоинта например: /User/sign-up
То есть итого запрос на сервер будет иметь следующий вид: https://kolab-plumboomteam-api.herokuapp.com/api/User/sign-up

Дальше идет список эндпоинтов и их сигнатуры

Эндпоинт для регистрации нового пользователя: /User/sign-up
Request Method: POST
Request Payload: {
  email: string
  firstName: string
  lastName: string
  password: string
  passwordConfirm: string
  role: "teacher" || "student"
}
Response: {
  message: "User was created"
}

Эндпоинт для авторизации пользователя: /User/sign-in
Request Method: POST
Request Payload: {
  email: string
  password: string
}
Response: {
  token: string
}

Эндпоинт для получения информации о пользователе: /User/user-info
Request Method: GET
Authorization: Bearer [token]
Response: {
  email: string
  firstName: string
  id: string
  lastName: string
  password: null
  role: "teacher" || "student"
}
